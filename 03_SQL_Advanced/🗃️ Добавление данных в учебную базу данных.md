# 🗃️ Добавление данных в учебную базу данных

## 📚 Практикум по операциям INSERT

В этом документе представлены примеры добавления данных в различные таблицы учебной базы.

---

## 🏗️ Структура базы данных
| Таблица | Описание |
|---------|----------|
| `📂 departments` | Отделы компании |
| `👥 workers` | Сотрудники |
| `🚀 projects` | Проекты |
| `✅ tasks` | Задачи |
| `🏷️ tags` | Теги задач |
| `🔗 tags_tasks` | Связь многие-ко-многим |
| `💰 salaries` | Выплаты зарплат |

---

## 📝 Примеры добавления данных

### 1. 🏷️ Добавление тегов
```sql
-- Добавить два тега в таблицу тэгов
INSERT INTO tags (name) VALUES 
('оптимизация'),
('документация');
```
Результат: В таблицу tags добавлены два новых тега для категоризации задач.

### 2. 🎨 Добавление сотрудников в отдел Дизайна
```sql
-- Добавить двух сотрудников в отдел Дизайна (department_id = 2)
INSERT INTO workers (department_id, name) VALUES 
(2, 'Петрова Мария'),
(2, 'Сидоров Артем');
```
Результат: В отдел дизайна добавлены два новых сотрудника.

### 3. 💻 Добавление сотрудников в отдел Программирования
```sql
-- Добавить двух сотрудников в отдел программирования (department_id = 1)
INSERT INTO workers (department_id, name) VALUES 
(1, 'Козлов Дмитрий'),
(1, 'Николаева Елена');
```
Результат: В отдел программирования добавлены два новых разработчика.

### 4. 🏢 Добавление новых отделов
```sql
-- Добавить два отдела
INSERT INTO departments (name) VALUES 
('Отдел сопровождения'),
('Отдел аналитики');
```
Результат: Созданы два новых структурных подразделения компании.

### 5. 🚀 Добавление новых проектов
```sql
-- Добавить три проекта
INSERT INTO projects (name) VALUES 
('Мир света'),
('Горизонт'),
('ПромСвязьДеталь');
```
Результат: В портфель компании добавлены три новых проекта.
